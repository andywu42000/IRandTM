- Tokenization.
- Lowercasing everything.
- Stemming using Porter’s algorithm.
- Stopword removal.
- Save the result as a txt file. 

### Text collection:
- One English news document  (https://ceiba.ntu.edu.tw/course/35d27d/content/28.txt ).
